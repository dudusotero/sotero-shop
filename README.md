This is an e-commerce project combining [Next.js](https://nextjs.org/) + [Swell](https://www.swell.is/).

## Getting Started

First, install all dependencies:

```bash
pnpm i
```

Copy `.env.example` file to `.env.local` and replace the variables with your own values.

```bash
cp .env.example .env.local
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

---

To run the production version, just run:

```bash
pnpm build && pnpm start
```
